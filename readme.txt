Automate adding the same page on multiple installs of a WordPress multisite.

== Description ==

This plugin works on multisite WordPress installs only.

Use this plugin when you need to add a page on multiple website installs of a WordPress multisite without doing this manually.

You have the following options for adding the page:  
* the list of sites you want to add the page to  
* page title  
* page slug  
* page content  
* set the page as Home, Blog or just simple page  
* set a page template  

In case your page already exists (has the same slug) on one of the websites, the page title and page content will be updated. 

Useful scenarios for this plugin:  

1. create a page in Visual Composer and deploy it to multiple sites in a multisite install  
2. set a page as home in multiple sites in a multisite install  
3. set a page as blog in multiple sites in a multisite install  

= Compatibility =

This plugin requires a multisite install.

== Installation ==

1. Copy the `plugin-multisite-create-pages` folder into your `wp-content/plugins` folder
2. Activate the Create Pages on Multisite WordPress plugin via the plugins admin page
3. Go to Tools->Create Pages on Multisite WordPress to open plugin page
